## What worked

- **Outbound bubble:** Lime `colors.primary` fill, `onPrimary` message text, tail corner — matches design intent (`ChatMessageBubble`).
- **Inbound bubble spec:** White `surface` + `border` (code) — should read clearly on gray chat background.
- **System messages:** Centered pill on `background` with `textSecondary` — separates automation from user voice.
- **Timestamps:** Inline bottom-right in bubble at 11px — doesn't dominate.
- **Game recap card:** Structured GAME RECAP block with attendance + streak — richer than plain text.
- **Quick-reply chips** above composer (`On my way`, `5 min late`) — reduces typing friction.
- **Game room header:** Collapsible game context bar + “Play needs you” banner — orients chat around session.
