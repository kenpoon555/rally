## What worked

- **Profile layout matches contract north-star:** setup lives on Profile; ongoing work on Today — both roles visible without a 5th tab.
- **Family row previews children** (“Alex · Beginner Badminton / Mia · …”) so parent context is one tap away.
- **Today blocks are labeled distinctly** — MY CLASSES (parent) vs CLASSES I TEACH (coach) with appropriate actions (child status vs Roster / Message Parents).
- **Coach Tools → Create Class** opens coach-only Class/Clinic flow while Family path remains separate.
- **Marcus seed** gives credible dual-role depth for demo/Validator (contrast with fresh-parent reviews).
