## What worked

- Single illustration system (`WelcomeTogetherIllustration`) scales cleanly at ~72% width.
- Three-slide narrative arc: discover → join game → crew/Rally.
- Slide 3 `MARKET_COPY.headline` is production-neutral (“Pickup sports and recurring crews near you.”).
- Pagination dots: active dot enlarges + primary color — clear position cue.
- Pill CTA (`radius.pill`, 56pt min height) matches auth stack elsewhere.

## Friction (prioritized)

| P | Screen | Issue | Suggested change | Contract impact |
|---|--------|-------|------------------|-----------------|
| P1 | Welcome footer | **Get Started** uses `colors.textInverse` (white) on `#B1E346` lime — weak contrast vs `theme.onPrimary` | Use `colors.onPrimary` (#141916) on lime CTA; audit all lime buttons | `module-visual-design-system` → CTA tokens |
| P1 | All slides | **Same illustration** on every slide — carousel feels static | Per-slide art crop or subtle motion OR drop pager to single hero | `module-visual-design-system` → welcome variants |
| P2 | Welcome footer | **“I have an account”** secondary link easy to miss below CTA; dev warning bar overlaps on sim | Increase footer reserved height; verify link contrast (`textSecondary`) | Welcome layout spec |
| P2 | Cold launch | **Invalid invite** alert from stale deep link — pollutes first frame | Clear pending invite on welcome mount when logged out | `flow-invite-to-rally` (behavioral; note in visual doc) |
| P3 | Dots | Inactive dots low contrast on `#EEEDEB` background | Slightly darker inactive dot or border | Spacing/color tokens |
| P3 | Slide 3 | `marketLine` caption repeats slide-3 body theme | Merge into body or style as subtle footnote only on final slide | Copy hierarchy |
