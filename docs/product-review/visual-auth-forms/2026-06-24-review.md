## What worked

- **Form card** pattern (`AuthScreenLayout`): white surface, 20px radius, border — reads as focused form island on `#EEEDEB` background.
- **TextField** rhythm: caption labels, 1.5px border, consistent vertical `spacing.lg` between fields.
- **Login CTA** “Continue to Rally” uses shared `Button` primary (`onPrimary` text) — **contrast correct** (unlike Welcome carousel CTA).
- **Sport chips** show 3 launch sports (Badminton, Pickleball, Basketball) — Build 12 wedge fix visible.
- **Age gate** options: large touch targets, selected state with primary border clear.
