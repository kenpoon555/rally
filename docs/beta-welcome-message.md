## Medium version (email / group intro)

**Subject:** Rally beta — host a game this week?

Hi everyone,

I'm opening a small beta of **Rally** — think *group chat for pick-up sports*, not a league app.

**How it works (2 minutes):**
1. Open **Rally** → **Discover** → **Create Game**
2. Pick your sport, set **time**, pick a **court/field** (or **Add a court** if nothing's listed)
3. Share the invite link or let nearby players request to join
4. Day-of: live in the **Game Room** chat — say hi, **Mark Ready**, host **Finalizes** roster
5. After the game: **Rate Players** on Profile; if you liked the crew → **Save as Regulars group** → **Schedule next game**

**Sports in this build:** Pickleball, Basketball, Badminton, Tennis, Volleyball, Soccer, plus Squash, Racquetball, Table Tennis, and Ultimate Frisbee for partner/crew sports that are hard to fill without an app.

**What I need from you:**
- Host **one** game OR join someone else's
- Tell me: Did chat + fixed time feel enough? Would you replay with the same group?

**Install:** [TestFlight / APK link]

Reply with your sport + neighborhood if you want me to seed courts near you.

Thanks,
[Your name]
